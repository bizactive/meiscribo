Intro

Last update: Jan '24.

The last couple of months I have focused on learning to code. This site aims to showcase some of those skills but also explores other learning that have taken place.

HTML and CSS is the path of least resistance, so I took it. JavaScript is a lot more challenging but I am relishing it, as I know it will simplify learning subsequent languages.

Why coding? A lifetime ago, I was a MS-SQL DBA and enjoyed writing stored procedures for the developers. But, after a time, I missed the hustle and bustle of callcenters, so decided to exit that path. Coding has been in the back of my mind though and considering what I will be doing in the next decade or 2, it makes to do it now.

This site uses the W3 CSS Framework in a grid layout. I will be tweaking this continously as time allows.

So...this will be kinda-like a blog, to keep HTML, CSS and JavaScript skills sharp. I'll highlight some work in my portfolio and discuss learnings on the other topics (especially where they cross) that are of interest to me. Here is a video changelog for this site.

I welcome LinkedIn connection requests to develop that network.

Coding

...I no longer see the code..

Cypher - The Matrix


The quote is overly-dramatic but quite effectve in the movie and indicative of how developers read code. They don't see a language made of notations - they read it as you are reading this English. So, that's my aim...

The sites used for this self-learning "escapade" are very good and I recommend them. However, if you have another resource... I'm all ears.

Learning from Tutorials, Exercises, Quizzes and Forums is not easy, but I have found that it reinforces knowledge and continously bears fruit with new approaches to problems and exploring the possibilities of the language/technology. Although, there is duplication, each of these sites has a different style of dissemenating knowledge that I found useful.

Kudo's to...

  • Mozilla Developer Network
  • scrimba
  • Stack Overflow
  • W3 docs


The plan is to get a couple of certificates, as soon as I can spend disposable cash on it, but here is some sense of the learning achieved:

I continously reference the material at W3Schools and want to pass the Web Development certification with them.

Although, I have several certificates from Sololearn, I want to tackle the exercises which are only available on subscription...for the sake of completion, but more as a super-huge thank you to the owners for the free learning that is offered. It would then be prudent to complete the Back-end and Data Science courses.

A delightful find is Code Combat where learning and play is combined for a thrilling experience. I'd like to get more involved with this community, as my son is an avid Roblox fan.

I am currently working my way through these exercises and learning React. So, what's next? ...polish off Front-end and then Back-end to reacquaint myself with SQL and learn Python et al.
